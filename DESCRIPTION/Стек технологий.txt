Вариант 1:
TypeScript - понятен в чтении. Удобно поддерживать в течении долго времени.
(требует транспиляции)

---------------------------------
Вариант 2:
Лучше использовать Rust с технологией WebAssembly (wasm)
Yew — молодой Rust фреймворк для создания фронтенд веб-приложений на WebAssembly

Frontend:
Yew - напоминает React. Клиентская часть
Cap'n Proto - (протокол), слой передачи данных приложения 
Backend:
actix-web - (веб-фреймворк, написанный на языке программирования Rust), 
Diesel ORM - для взаимодействия с PostgreSQL

---------------------------------
Вариант 3:
Backend: actix-web 
BD: PostgreSQL
Frontend: TypeScript, WASM (wasm-bindgen или yew)
Маршрутизация: OpenStreetMap 
Аутентификация и авторизация - JWT (JSON Web Tokens) на основе создания токенов доступа
Развёртывание: Docker
Мониторинг производительности приложения: Prometheus и Grafana
